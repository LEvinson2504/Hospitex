## Local Setup

### React Native Setup

1. Install depedencies

```bash
cd client
yarn
```

2. Start local server with Expo

```bash
yarn start
```

### Node.js + Express Setup

1. Install depedencies

```bash
cd server
yarn
```

2. Setup env files with the example at .env.example
3. Start the server

```bash
yarn start
```
